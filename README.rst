My personal configs.

Variable and template
---------------------

Variables should be managed in ``~/.shenv``.
Python templates use environment variables in ``~/.shenv``.

How to use
----------

Just clone this repo to your home directory.

.. code:: git clone https://github.com/florious0721/dotfiles.git ~
